# 💖 เว็บไซต์บอกรักแฟนแสนพิเศษ (Interactive Love & Candle Blowing Cake Web App)

ยินดีด้วยครับ! เว็บไซต์บอกรักแฟนสุดพรีเมียมของคุณได้รับการสร้างและจัดระบบอย่างสมบูรณ์แบบเรียบร้อยแล้วในโฟลเดอร์นี้

---

## 🌟 ฟีเจอร์ที่ยอดเยี่ยมภายในเว็บไซต์
1. **หน้าเค้กเป่าเทียนไขแบบ Interactive:** 
   - เทียนไข 3 เล่มปักอยู่บนเค้ก 3D CSS โดยเปลวเทียนจะสั่นไหวและดับลงทีละเล่มเมื่อคุณเป่าลมใส่ไมโครโฟนจริง!
   - มีปุ่มสำรองแสนน่ารัก **"เป่าเทียนด้วยรัก 🎂"** เผื่อแฟนเปิดบนอุปกรณ์ที่ไม่สะดวกแชร์ไมโครโฟน
   - เมื่อเทียนดับหมด จะเกิดการพุ่งกระจายอย่างอลังการของ **Confetti หัวใจและประกายวิบวับระเบิดเต็มหน้าจอ** วาดบน Canvas 60 FPS
2. **แกลเลอรีแสนโรแมนติก (Polaroid Grid):**
   - แสดงรูปภาพความทรงจำ 5 ภาพจากโฟลเดอร์ `MuMu` ของคุณที่ได้รับการแปลงไฟล์ HEIC และบีบอัดให้แสดงผลบนเบราว์เซอร์ได้รวดเร็ว (โหลดทันใจ)
   - กรอบรูปโพลารอยด์หมุนกระจายดูธรรมชาติพร้อมคำบอกรักที่ลึกซึ้ง และมีเอฟเฟกต์หมุนตั้งฉากเมื่อเลื่อนมาชม (Scroll Reveal)
3. **จดหมายรักประทับตราครั่ง (Wax-Sealed Love Letter):**
   - มีซองจดหมาย 3D ที่ถูกปิดผนึกไว้ด้วย**ตราครั่งหัวใจสีแดง** 
   - เมื่อแฟนกดคลิกที่ตราครั่ง ครั่งจะแตกออก ฝาซองเปิดขึ้น และมีจดหมายรักแบบแก้วใส (Glassmorphism) เลื่อนขึ้นมา พร้อม**แอนิเมชันการพิมพ์ข้อความบอกรักทีละอักษร**แสนอบอุ่น
4. **เครื่องเล่นเพลงอัจฉริยะ (Lover's Player):**
   - เล่นดนตรีสดที่สร้างจากโค้ด (Procedural Synthesizer) โดยใช้ Web Audio API เล่นแนวคอร์ด Lofi สดๆ แบบออฟไลน์ไม่มีวันสะดุด
   - และสามารถคลิกปุ่มสลับเป็นเพลงบรรเลงโรแมนติกที่มุมบนขวาได้ทันที

## 🚀 วิธีการเปิดใช้งานบนเครื่องของคุณ (How to Run)

คุณไม่จำเป็นต้องติดตั้งโปรแกรมหรือโมดูลใดๆ เพิ่มเติมเลย เพราะเราเขียนเซิร์ฟเวอร์ด้วย **Pure Node.js** ซึ่งมีติดมาในเครื่องอยู่แล้วครับ!

### ขั้นตอนการรัน:
1. เปิดโปรแกรม **Terminal** ในเครื่อง Mac ของคุณ
2. รันคำสั่งต่อไปนี้เพื่อไปยังโฟลเดอร์งาน (หรือเปิด Terminal ที่โฟลเดอร์นี้โดยตรง):
   ```bash
   cd /Users/sittisakthongsri/Desktop/MuMu
   ```
3. รันคำสั่งเพื่อสตาร์ทเว็บไซต์:
   ```bash
   node server.js
   ```
4. เมื่อเซิร์ฟเวอร์เปิดขึ้น คุณสามารถเข้าชมผลงานบนเครื่องของคุณเองได้ทันทีผ่านลิงก์:
   👉 **[http://localhost:3000](http://localhost:3000)**

## 🔗 วิธีทำให้ผู้อื่น / แฟนสามารถเข้าใช้งานผ่านลิงก์ได้ทันที (Public Sharing)

หากคุณต้องการส่งลิงก์ให้แฟนเข้าชมจากมือถือของเธอ หรือจากที่อื่นตอนที่เว็บกำลังรันอยู่บนเครื่องคุณ มีวิธีแชร์ผ่านอุโมงค์สัญญาณอินเทอร์เน็ตที่ฟรีและรวดเร็วอยู่ 2 วิธี (เลือกวิธีใดวิธีหนึ่งครับ):

### วิธีที่ 1: ใช้ Pinggy (แนะนำเป็นพิเศษ - ไม่ต้องติดตั้งโปรแกรมอะไรเลย!)
ใช้ความสามารถของ SSH ที่มีอยู่ใน macOS เพื่อดึงลิงก์สาธารณะโดยไม่ต้องลงโปรแกรมเพิ่ม:
1. เปิดหน้าต่าง **Terminal ใหม่ขึ้นมาอีกอันหนึ่ง** (โดยไม่ปิดหน้าเซิร์ฟเวอร์หลักที่รันอยู่)
2. คัดลอกและวางคำสั่งนี้ลงไปใน Terminal แล้วกด Enter:
   ```bash
   ssh -R 80:localhost:3000 a.pinggy.io
   ```
3. Terminal จะแสดงคำตอบและให้ลิงก์ **HTTPS** แก่คุณ (เช่น `https://rxxxx.a.pinggy.link`)
4. **คัดลอกลิงก์ HTTPS นั้นไปส่งให้แฟนของคุณได้ทันที!** เธอกดเปิดอ่านบนมือถือจากที่ไหนก็ได้ในโลกเลยครับ 💖

### วิธีที่ 2: ใช้ Localtunnel
หากเครื่องคอมพิวเตอร์ของคุณมี Node.js และ npm ติดตั้งอยู่แล้ว สามารถรันสิ่งนี้ได้โดยตรง:
1. เปิดหน้าต่าง **Terminal ใหม่ขึ้นมาอีกอันหนึ่ง**
2. รันคำสั่งนี้:
   ```bash
   npx localtunnel --port 3000
   ```
3. ระบบจะสร้างลิงก์สาธารณะที่หน้าตาประมาณ: `https://xxxxxx.localtunnel.me`
4. เมื่อแฟนคลิกลิงก์นี้เป็นครั้งแรก ระบบอาจจะขึ้นหน้าจอเพื่อความปลอดภัย ให้คลิกปุ่ม **"Click to Continue"** หรือใส่ IP เครื่องของคุณ (ซึ่งตัวเว็บ Localtunnel จะบอกไว้ในจอ) เพื่อเปิดเข้าใช้งานหน้าเว็บหลักได้เลยครับ!
